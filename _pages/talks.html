Dataset

Job Shop scheduling problem

  • JSP from github link ABZ5-9(Adams), FT06, FT10 and FT20(Fisher and Thompson), LA01-40(Lawrence), ORB01-10(Applegate and Cook), SWV01-20 (Storer), yn1-4 (Yamada and Nakano), ta01-80 (Taillard)
  • JSP, FJSP, FJSP_SDST,FSP,FAJSP from github link Job Shop Scheduling Benchmark: Environments and Instances for Learning and Non-learning Methods (RobbertReijnen)

Flexible Job Shop scheduling problem

  • FJSP Mk01-10(Brandimarte), DP01-18(Dauzere), Barnes,Hurink
  • FFJSP FJSP with triangular fuzzy processing time, Lei01-05(Deming. Lei), Gao01-08(Kaizhou Gao)
  • T2FJSP FJSP with type-2 fuzzy processing time, Li01-30(Junqing. Li)
  • DHFJSP Distributed heterogeneous FJSP containing 20 instances and a real world case(Rui. Li)
  • FJSP-T FJSP with finite transportation resources, containing 50 instances from two literature(Bilge and Ulusoy (1995), Deroussi and Norre (2010))

Permutation Flow Shop scheduling problem

  • Taillard-PFSP-1993(120 instance) Permutaion flowshop problem, (Taillard.E)
  • VFR-PFSP-2017(240 instance) Permutaion flowshop problem, (Eva Vallada)
  • DPFSP Distributed permutaion flowshop problem, (Junqing. Li)
  • DPFSP(480 small, 720 large) Distributed permutaion flowshop problem (most use), (Naderi, B 2010)
  • DAPFSP Distributed assambly flowshop problem, (Junqing. Li)
  • PFSP_NOIDLE Permuation flowshop problem with no idle time
  • FlowShop_Mixed_No_Idle Flow shop problem with mixed no-idle time
  • PFSP_SDST Permuation flowshop problem with sequence dependend set up time
  • DHPFSP Distributed heterogeneous PFSP (Rui.Li)

Hybrid Flow Shop scheduling problem

  • HFSP Hybrid flowshop problem, Li01-77(Junqing. Li)
  • EADHFSPEnergy-Aware Distributed Hybrid flow shop problem with uniform machine(Jingjing. Wang, TEVC,2022)
  • DHHFSPMPJ Distributed heterogeneous HFSP containing 20 instances and a real world case(Rui. Li)

Cplex code for MILP model

  • EEFJSP Energy-efficient flexible job shop scheduling problem
  • EEDFJSP Energy-efficient distributed flexible job shop scheduling problem
  • CPLEX-DHHFSPMPJ Energy-efficient distributed heterogeneous hybrid flowshop scheduling with multiple priorities of job (Rui.Li)
